# Contributing

Issues and pull requests are welcome after the repository is public.

Please preserve the scope boundary:

- Do not add claims of first-principles derivations unless supported by derivations, tests, and documentation.
- Run `ruff check .`, `ruff format --check .`, `mypy src`, `pytest -q`, `python -m build`, and `python scripts/generate_report.py` before submitting changes.
- Keep generated artifacts clearly separated from source where possible.
